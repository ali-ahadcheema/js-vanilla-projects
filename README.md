# JavaScript Projects

This repository contains beginner projects built using **HTML, CSS, and Vanilla JavaScript**.
These projects helped me practice API integration and DOM manipulation.

## Projects

### 1. Weather App

The Weather App shows the **current weather and forecast for cities in Pakistan**.

Features

* Search weather by city name
* Shows temperature and weather conditions
* Uses a weather API to get real-time data

### 2. Currency Converter

The Currency Converter allows users to **convert one currency into another** using live exchange rates.

Features

* Convert between different currencies
* Real-time exchange rate data
* Simple user interface
